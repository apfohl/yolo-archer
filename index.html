# Einführung
## Linksammlung: * Übersichtliche Zusammenfassung von Rechercheergebnissen * Aktuell * Inhalt der Quelle
## Bisherige Lösungen: * Citavi (http://citavi.com/de/) * jabref (http://jabref.sourceforge.net/) * Firefoxplugin Zotero (http://www.zotero.org/) * Textdatei in der Cloud
## Probleme: * Sind Einzelplatzlösungen * Setzen spezielle Software oder Plugin-Installation vorraus * Keine Bewertungen * Kein Multiuser, kein verteiltes Arbeiten
# Unsere Idee ### Webdienst angelehnt an [StackOverflow](http://stackoverflow.com/) * Keine Softwareinstallation * Quelle, Author, Tags * Notizen * markdown * Bewertungen (Quelle und Notizen) * Übersichtlicher Aufbau * Multiuser
# Implementation
* Twitter Bootstrap * Ruby on Rails * PostgreSQL * Deploy auf Heroku per Git Deployment: 

	git clone https://github.com/apfohl/yolo-archer.git
	heroku create
	git push heroku master
